## Setup

Check out this repository and then run,

```console
$ bundle install
$ bundle exec rails db:migrate
$ bundle exec rails server
```

## Features

Here are the features that are include:

- User Register capabilities
- Login capabilities
- Get all Tweets
- Create new Tweet
- Delete a tweet
- Update a tweet
